Realtime Crowd Insights

Realtime Crowd Insights is a mobile app for iOS that obtains demographic information from the analysis of people in images using Azure's facial recognition API.

Background

This project was developed for Blue People, a company specialized in building software, with the intention of exploring the scope and limitations of Azure's Face API, so that Blue People can eventually offer this service to their customers.

Tools

  • Swift
  • Azure Face API
